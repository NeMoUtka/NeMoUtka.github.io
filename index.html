Architecture

How It Works

A four-stage pipeline transforms raw camera frames into structured defect reports in real time.

01
📷
Image Capture

High-resolution images are acquired from industrial cameras or uploaded datasets for offline evaluation.

02
⚙️
Preprocessing

Normalization, resizing to 640×640, and augmentation (flip, crop, noise) prepare inputs for the backbone.

03
🧠
Model Inference

A fine-tuned YOLOv8 / CNN backbone runs forward pass, producing bounding boxes and class probabilities.

04
📊
Post-processing

NMS filters overlapping boxes; results are visualized with color-coded overlays and logged to a report.

The Project

About

Project Overview

This project applies deep learning to automated quality control. Traditional manual inspection is slow and error-prone; our model provides consistent, real-time analysis that generalises across surface types and lighting conditions.

Dataset & Training

The model was trained on the NEU Surface Defect Database, augmented with custom images. Training ran for 100 epochs on a single NVIDIA GPU using PyTorch, with early stopping and cosine LR scheduling.

Tech Stack

Python · PyTorch · YOLOv8 · OpenCV · scikit-learn · Weights & Biases for experiment tracking · ONNX for model export and edge deployment.
